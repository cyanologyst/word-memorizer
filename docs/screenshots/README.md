# Product polish screenshots

The screenshot sets document the July 2026 production-polish pass.

- `product-polish-before/` contains the last complete historical UI audit captured before this pass.
- `product-polish-after/` contains the accepted screenshots from the final live-app audit at 1426 x 836, plus minimum-window checks at approximately 746 x 633.

The after set covers Dashboard, Review setup, active/revealed/paused/completed review states, Mistake Lab, Wordlists, Statistics, Activity, Achievements, Logs, Settings, About, the command palette, keyboard shortcuts, and representative minimum-width layouts.

The UI audit was non-destructive. A one-word review session was completed to verify the keyboard flow, then the original progress, achievement, and JSONL review-history files were restored from a temporary backup before the final screenshots were captured.
